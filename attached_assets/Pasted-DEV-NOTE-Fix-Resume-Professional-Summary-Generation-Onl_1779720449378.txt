Goal

Fix only the Professional Summary section.

Do not touch anything else.

The title is now okay.

Only fix how the resume summary is generated.

Problem

The current summary is still being written like a job description keyword dump.

Bad example:

Roofing Labourer / Crew Support — installation-focused candidate with strengths in safety procedures, heavy lifting, team work, and on-site problem solving...

This is wrong because it sounds like the AI is summarizing the job being applied to.

The Professional Summary should summarize the applicant, not the job.

Correct Rule

The Professional Summary must be based on the applicant’s actual resume, work history, skills, and previous experience.

The job description should only be used to decide which parts of the applicant’s real background are most relevant.

Do not invent experience.
Do not claim the applicant did job duties they never did.
Do not turn the job description into the summary.
Do not write “installation-focused candidate” unless the applicant actually has installation experience.

Main Instruction

Rewrite the resume summary generator using this logic:

Read the applicant’s existing resume/profile first.
Extract real previous roles.
Extract real responsibilities.
Extract real transferable skills.
Compare those real experiences to the target job.
Write a short professional summary about the applicant.
Only include job-related wording when it connects naturally to the applicant’s real experience.

The summary should answer:

Who is this person?
What work experience do they actually have?
What useful skills do they bring?
Why does their background make sense for this job?

It should not answer:

What is this job description asking for?

Required Summary Formula

Use this structure:

Sentence 1:
Start with the applicant’s real background.

Sentence 2:
Mention actual previous responsibilities and strengths.

Sentence 3:
Connect those strengths to the job being applied for.

Optional sentence 4:
Mention reliability, teamwork, safety, communication, or attention to detail if supported by the resume.

Unless the resume clearly says the applicant has done roofing work.

Do not generate:

Unless the applicant’s profile clearly proves that.

Do not generate:

Unless the applicant actually has that experience.

Do not generate:

That is not a resume summary.
