Fix Import Syntax Properly

Do NOT guess. Use safe compatible pattern.

If project uses CommonJS:

Replace:

import pdfParse from "pdf-parse"

WITH:

If project uses ESM:

import pdfParse from "pdf-parse"

import pdfParse from "pdf-parse"

AND ensure you call it correctly:

const pdfParse = pdfParseModule.default || pdfParseModule
